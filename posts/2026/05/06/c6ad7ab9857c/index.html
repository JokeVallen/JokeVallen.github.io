C#比较器全局工具类

更新于2026-05-06|C#
|浏览量:

前言

出于全局管理比较器的思路，在应用中常见需求，需要统一提供比较器替换的入口，统一调用的入口，以提供更灵活的比较策略，以及针对非泛型比较器的获取支持，我们设计了这个比较器全局工具类，包括 IComparer、IComparer<T>、IEqualityComparer、IEqualityComparer<T> 四种 C# 提供的比较器接口支持，它允许你为任意类型注册自定义比较器，之后无论是在泛型还是非泛型上下文中获取，都能得到同一实例，并自动适配只实现了单一接口的比较器，同时内置了默认比较器的反射缓存和完整的缓存清理机制，从而在整个应用程序中实现比较逻辑的统一替换与高效复用。

公共 API 说明

ComparerUtility 提供相等性比较器与排序比较器的获取、设置、移除、清空四类操作，每类均同时支持泛型与非泛型调用。
