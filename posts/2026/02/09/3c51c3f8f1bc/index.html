前言

在项目开发过程中，有时候我们想要监听某个组件是否添加或者删除，Unity 本身并没有提供这类 API，不过我们可以尝试通过 API 组合来寻找解决方案。本篇文章将作为不定期更新文章，后续可能提供更多的解决方案。

解决方案

ObjectFactory + ExecuteInEditMode + OnDestroy

Awake + OnDestroy

方案对比

方案 适用场景 适用时机
ObjectFactory + ExecuteInEditMode + OnDestroy 该方案适用于通过 Inspector 面板添加和删除组件的行为监听。 编辑器时
Awake + OnDestroy 该方案适用于通过AddComponent和Destroy或DestroyImmediate方法增删组件的行为监听。 运行时

Unity常识篇（一）
Awake 与 OnDestroy 的关系当组件被添加到活动状态的游戏对象时，Awake 会被调用，当组件被销毁时，OnDestroy 会被调用。如果 Awake 未被调用，那么 OnDestroy 也不会被调用，例如将组件添加到禁用状态的游戏对象然后再销毁组件时，二者都不会被调用。
